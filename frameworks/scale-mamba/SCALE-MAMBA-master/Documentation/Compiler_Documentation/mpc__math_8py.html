◆ log2_fx()

def Compiler.mpc_math.log2_fx (   x)

Returns the result of log_2(x) for any unbounded number.

This is achieved by changing x into f*2^n where f is bounded by[0.5, 1]. Then the polynomials are used to calculate the log_2 of f, which is then just added to n. Note that sfloat and sfix sizes have to be parametrized correctly, such that sfix vlen > sfloat vlen. This is not the case by default.

Parameters
xinput for log_2 (sfix, sint).
Returns
returns (sfix) the value of log2(X)
